# Java OOP Complete Roadmap

## Introduction
This repository is designed for learning Java Object-Oriented Programming (OOP) from basics to advanced concepts.  
It covers:

- Java Basics
- Classes and Objects
- Constructors
- Inheritance
- Polymorphism
- Abstraction
- Encapsulation
- Interfaces
- File Handling
- Mini Project (Student Management System)

The goal is to build a strong foundation in Java OOP and real-world programming.
